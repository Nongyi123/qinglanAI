# qinglanAI
涵盖 5 个预设模板（适配不同分析场景），均包含明确的分析维度、数据要求和输出导向，可直接用于 AI 分析工具：
一、基础行情解读模板（核心：趋势 + 关键位 + 形态）
请综合分析当前 K 线图的走势，明确短期的趋势方向（上涨、下跌、震荡）。请指出关键的支撑位和阻力位，以及可能出现的反转形态。
适配数据：当前周期 K 线数据（开盘 / 收盘 / 最高 / 最低）、基础技术指标（MA 均线）
二、进阶交易机会挖掘模板（核心：入场 + 止损 + 止盈 + 突破判断）
请根据当前周期的 K 线图分析最佳的交易机会，给出具体的入场点、止损点和止盈点。分析当前趋势是否处于突破 / 跌破关键位置，并给出交易策略。
适配数据：当前周期 K 线数据、支撑 / 阻力位数据、价格波动区间
三、纯技术指标分析模板（核心：MACD+KDJ + 均线系统信号）
请重点分析 MACD、KDJ 和均线系统（MA5, MA10, MA20）当前 K 线图的状态。根据指标金叉、死叉、背离等信号，给出后市的预测。
适配数据：K 线数据 + MACD（DIF/DEA/MACD-BAR）、KDJ、MA5/MA10/MA20 均线数据
四、当前持仓风险评估模板（核心：持仓利弊 + 调整建议）
假设我已经持有该币种多头 / 空头仓位，请评估当前 1 小时 K 线形态下的持仓风险和潜在收益，并建议是否应该部分止盈、加仓或止损。
适配数据：当前周期 K 线数据、实时价格、波动幅度、技术指标信号
五、日内短线交易模板（核心：24 小时内短线策略 + 风险等级）
仅关注当前 K 线图的走势，给出当日内（24 小时内）最有可能发生的行情变动和短线交易建议，要求快进快出，并明确风险级别（低 / 中 / 高）。
适配数据：日内 K 线数据、短期波动规律、实时成交量
补充：AI 分析的默认数据支撑（HTML 隐含逻辑）
所有提示词均默认结合以下数据维度，无需额外添加：
实时同步的交易所 K 线数据（Binance 数据源，每 5 秒更新）；
全球加密行业资讯（消息面）；
链上实时指标（链上数据异动）；
多周期可选（15m/1h/4h/1d，支持单周期或全周期交叉分析）。
